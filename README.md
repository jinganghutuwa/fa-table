# fs-table
## 项目说明：
1. 下载本项目后需自行新建excel,html两个文件夹；
2. excel文件夹用来存储所要转化html文件的Excel表格文件
3. html文件夹是用来存储Excel表格转化之后的HTML文件的



## 项目启动及构建
1. 下载依赖：npm install
2. 启动：npm run serve
